### 🛠️ Selected Projects

#### 🏥 Medical Intelligent QA System
> **核心标签：RAG / Re-ranking / Multi-modal**
- **挑战**：解决医疗问答中的幻觉问题与专业术语对齐。
- **方案**：集成多级检索增强、语义重排序 (Rerank) 与对话缓存，支持多模态输入解析。
- **效果**：显著提升了复杂医疗咨询场景下的回答准确度与上下文关联性。
